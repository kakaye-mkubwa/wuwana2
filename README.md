# Wuwana v2

https://www.wuwana.com

## How to test locally

Open a Terminal and clone the project source with Git:  
`git clone https://github.com/wuwanahq/wuwana2.git`

Go in the project directory and run the PHP built-in web server:  
`php -S localhost:8000`

Then open open open your browser and go to http://localhost:8000  
To stop the PHP web server press Ctrl+C in the Terminal.
